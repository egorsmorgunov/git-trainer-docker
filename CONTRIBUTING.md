## Contribution
Before contribute please check errors and codestyle.

```
git clone …
npm install
vi …

#build
./node_modules/.bin/grunt start --target=dev

#check errors and codestyle
./node_modules/.bin/grunt lint

#fix codestyle ;)
vi …

#then commit and do pull-request
git commit …
```
